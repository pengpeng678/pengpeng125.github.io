3.1 安装 Squirrel

一个可以访问 Phoenix 的图形化工具.

步骤 1: 下载 Squirrel

http://squirrel-sql.sourceforge.net/

步骤 2: 选择 java 打开方式

由于下载的是一个可执行 jar, 很多 windows 操作系统默认用压缩工具打开, 需要手动改成用 java 打开
