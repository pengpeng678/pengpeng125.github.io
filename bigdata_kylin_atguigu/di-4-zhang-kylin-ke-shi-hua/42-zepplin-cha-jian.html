4.2 Zepplin 插件

一个可视化工具. 用来从 Kylin 查数据

步骤1: 安装和启动

/opt/software/kylin » tar -zxvf zeppelin-0.8.0-bin-all.tgz -C /opt/module/

/opt/module » mv zeppelin-0.8.0-bin-all zeppelin

/opt/module/zeppelin » bin/zeppelin-daemon.sh start
Log dir doesn't exist, create /opt/module/zeppelin/logs
Pid dir doesn't exist, create /opt/module/zeppelin/run
Zeppelin start                                             [  OK  ]

登录 web 页面查看: http://hadoop201:8080
