第 2 章 Kylin 环境搭建

2.1 从官网下载需要的安装包

  1. 官网地址

    http://kylin.apache.org/cn/

  2. 官方文档

    http://kylin.apache.org/cn/docs/

  3. 下载地址

    http://kylin.apache.org/cn/download/

    我们选择2.5.1版本

2.2 安装部署

步骤1: 上传安装包到 linux

apache-kylin-2.5.1-bin-hbase1x.tar.gz

步骤2: 解压安装包

tar -zxvf apache-kylin-2.5.1-bin-hbase1x.tar.gz -C /opt/module/

步骤3: 检查环境变量

需要用到如下环境变量:

/opt/software/kylin » echo $HIVE_HOME
/opt/module/hive-1.2.1

/opt/software/kylin » echo $HBASE_HOME
/opt/module/hbase-1.3.1

/opt/software/kylin » echo $HADOOP_HOME
/opt/module/hadoop-2.7.2

如果没有配置, 需要专门配置下, 并使其生效.

步骤4: 修改conf/keylin.properties中的时区配置

# 改成东八区, 否则会出现时间显示不准的问题
kylin.web.timezone=GMT+8

步骤5: 启动 Kylin

启动 Kylin 之前需要先启动 HDFS, Yarn, Zookeeper, HBase 
bin/kylin.sh start

步骤6: 登录 Kylin Web

http://hadoop201:7070/kylin/login

登录名: ADMIN 密码:KYLIN 注意: 都是大写

可能出现的问题解决方案

  • 如果有报错: java.net.UnknownHostException: hadoop201:2181: 未知的名称或服务. 需要把 HBase 的hbase-site.xml 中的关于 Zookeeper 的配置中的端口号2181去掉.

      <!-- 配置 Zookeeper -->
  <property>
      <name>hbase.zookeeper.quorum</name>
      <value>hadoop201,hadoop202,hadoop203</value>
  </property>

  • 如果报错: Failed to find metadata store by url: kylin_metadata@hbase, 则可以试着把 hdfs 中的/hbase目录删除, 和 zookeeper 下的/hbase节点删除. 然后重新启动 HBase, 再启动 kylin
