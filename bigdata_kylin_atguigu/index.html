第 1 章 Kylin 概述

Apache Kylin™是一个开源的分布式分析引擎，提供 Hadoop/Spark 之上的 SQL 查询接口及多维分析（OLAP）能力以支持超大规模数据

OLAP（Online Analytical Process），联机分析处理, 或者在线分析

最初由eBay Inc. 开发并贡献至开源社区。

它能在亚秒内查询巨大的 Hive 表。

Apache Kylin也是中国人主导的、唯一的Apache顶级开源项目，在开源社区有世界级的影响力。
